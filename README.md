Pour lancer notre boutique en ligne, voici les commandes:

En local sur votre terminal:
1. ``` $ git clone https://github.com/ClementPain/cats_project ```
2. ``` $ bundle install ```
3. ``` $ rails db:drop ```
4. ``` $ rails db:create ```
5. ``` $ rails db:migrate ```
6. ``` $ rails db:seed ```
